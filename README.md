## Student-Career-Recommendation-using-ANN
This project uses an Artificial Neural Network (ANN) to recommend suitable career paths for students based on their academic performance, skills, and interests. It helps students make data-driven career decisions by analyzing patterns and predicting the best-fit career options.
## Objective
To build an intelligent system that suggests the best career options for students using machine learning.
## Features
Takes student data (marks, skills, interests)
Uses ANN model for prediction
Provides career recommendations
Easy to use and understand
## Technologies Used
Python
NumPy
Pandas
Scikit-learn
TensorFlow / Keras
Google Colab
## How It Works
Input student details
Data preprocessing
Train ANN model
Predict career path
Display result
##  How to Run
Open the .ipynb file in Google Colab
Run all cells step by step
Enter input values
View the predicted career
## Future Scope
Add more career options
Improve model accuracy
